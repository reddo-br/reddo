Reddoでは、データディレクトリ下のscriptsディレクトリにrubyスクリプトを置くことで、動作を制御できます。読み込みには再起動が必要です。

scriptsディレクトリは、

%APPDATA%\reddo\scripts (windows)
~/Library/Application Support/reddo/scripts (mac)
~/.reddo/scripts (linuxなど)

などにあります。
「設定」ページの一番下にユーザーディレクトリ/フォルダの位置が表示されています。その中のscriptsディレクトリ/フォルダがそれです。

現在、ignore判定と、サムネイル取得urlの指定ができます。

sampleスクリプトのコメントを参考にしてください。
